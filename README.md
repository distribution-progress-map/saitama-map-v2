# saitama-map-v2

埼玉県知事リコール周知用チラシの配布進捗共有マップです。

## 概要

市町村ごとのチラシ配布エリアを地図上で共有・管理するためのWebマップです。
Leaflet・Google Apps Script・Googleスプレッドシートを利用して運用しています。

---

## ページ構成

### index.html

埼玉県全体の確認用ページです。

* 市町村境界を表示
* 市町村クリックで各市町村ページへ移動
* 将来的には軽量サマリー表示用ページとして運用予定

---

### map.html

市町村共通の配布エリア管理ページです。

URLパラメータで対象市町村を切り替えています。

例：

map.html?city=川越市

主な機能：

* 配布エリア作図
* 配布予定／配布済管理
* 修正依頼
* 管理者連絡
* ID検索
* スマホ対応
* 2026/5/26　LINEオプチャ内のリンクから開くと不具合が起きるため、修正対応

---

## データ管理

配布エリア情報は Google スプレッドシートで管理しています。

Apps Script 側で市町村フィルタを行い、各市町村ページでは対象市町村のデータのみ取得する構成になっています。

---

## ファイル構成

* index.html
  埼玉県全体ページ

* map.html
  市町村共通ページ

* saitama.geojson
  埼玉県市町村境界データ
